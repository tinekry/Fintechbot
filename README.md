# nlogn-bank

Telegram-бот банка и биржи труда для лагерной игры «День миллионера».

## Быстрый запуск

1. Установите Python 3.11 или новее.
2. Создайте бота через `@BotFather` и получите токен.
3. Скопируйте `.env.example` в `.env` и заполните:
   - `BOT_TOKEN`;
   - `MAIN_ADMIN_USERNAMES` — username главных организаторов через запятую, без `@`;
   - `STAFF_LOG_CHAT_ID` — ID закрытого чата педсостава (бот должен быть в нём).
4. Поместите whitelist в `data/members.csv`. Заголовки должны быть:
   `Name,User ID,Phone,Username,Last Seen,Bio`.
5. Выполните:

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python bot.py
```

Главные администраторы должны присутствовать в CSV и иметь заполненный username.

## Важные правила

- Все суммы — целые положительные nlogn-коины.
- Создавать задачи могут администраторы и назначенные ими работодатели.
  Работодатели не могут выдавать этот доступ другим. Лимит обычного
  работодателя — 10 незакрытых задач, проверенного работодателя — 50.
- Задача может быть для одного или нескольких исполнителей. У массовой задачи
  первый отчёт замораживает приём остальных до решения работодателя.
- Создатель может отменить задачу в течение минуты; администратор — в любое время.
- Награда задачи холдируется на счёте любого работодателя, включая администратора.
- Денежные операции и взятие задач атомарны: повторное нажатие не вызывает
  двойного списания.
- Полный журнал денежных операций сохраняется в `data/transactions.csv`;
  `@tinekry_u` может скачать его из панели государства.
- FSM-сценарии хранятся в памяти. После перезапуска незаконченный диалог нужно
  начать заново через `/menu`; деньги, задачи и отчёты сохраняются в SQLite.
- Для продакшена запускайте один экземпляр бота с одной SQLite-базой.

## Команды

- `/start` — авторизация и первое приветствие;
- `/menu` — сброс текущего диалога и возврат в главное меню.

## Выборы

Кнопка «🗳 Выборы» открывает список кандидатов, регистрацию и собственную
анкету. Анкета включает программу и 1–5 фотографий, после чего отправляется
главным администраторам на модерацию. Один Telegram-аккаунт может отдать
только один голос за весь период выборов.

В разделе «Моя анкета» кандидат может добавить фотографии, видео и аудио
(до 15 файлов суммарно, фотографий — до 5) или удалить отдельные вложения.
После редактирования анкета повторно проходит модерацию; для публикации в ней
должна остаться хотя бы одна фотография.

Голос сохраняется при редактировании анкеты, а проголосовавшие получают
уведомление об изменениях. Пункт «Мой голос» показывает выбранного кандидата и
позволяет отменить голос. При отклонении или снятии кандидата его избиратели
уведомляются и могут проголосовать повторно.

В панели главного администратора раздел «Управление выборами» позволяет снять
кандидата (вместе с его голосами) или изменить отображаемый счётчик голосов по
числовому ID кандидата.

Административные действия сохраняются в `data/admin_actions.log` и, если
настроен `STAFF_LOG_CHAT_ID`, одновременно отправляются в служебный Telegram-чат.
